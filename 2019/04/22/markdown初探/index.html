markdown是什么？

Markdwon 是一种轻量级 标记语言，它以纯文本形式（易读、易写、易更改）编写文档，并
最终以HTLM格式发布。Markdown也可以理解为将以 MARKDOWN语法编写的语言转换成HTML内容的工具。

注意事项

标记符号(#,+,*….)后面要加空格或者Tab, 不在区域内的要加一个空行分隔
